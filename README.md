# Maintainability in Robotics Software Systems: A Systematic Literature Review
### Replication Package
This repository contains the data extracion sheet, the plots and the .bib file of references used for the SLR conducted in the BSc thesis "Maintainability in Robotics Software Systems: A Systematic Literature Review" by Valentin Stefanescu, University of Groningen, 2026.

##Overview
This SLR reviews 52 primary studies on maintainability in robotics software, covering five main research questions (RQ1–RQ5, with sub-questions RQ2.1, RQ5.1, RQ5.2). Full methodology is described in Section 4 of the thesis.
